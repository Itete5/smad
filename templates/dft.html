Density functional theory

Configure and launch DFT calculations (e.g. Quantum ESPRESSO, VASP, ABINIT) for electronic structure, total energy, and properties of superconducting candidates. Link structures from the Structures page and inspect results in Analysis.

Input / structure

Select a structure from your database or paste CIF/POSCAR. Set k-mesh, cutoff, functional.

Code / backend

Choose code (QE, VASP, …), pseudopotentials, and HPC queue. Submit via Cluster Console or API.

Run & monitor

Launch jobs, track status, and stream logs. Results feed into Band gap, DOS, and Charge density in Analysis.

Post-process

Band structure, DOS, charge density, and convergence checks. Export for publications.
